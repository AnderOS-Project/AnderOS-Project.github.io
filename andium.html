Andium

The modular browser built for speed, packs, and slash commands.

What is Andium?

Andium is a lightweight browser project designed by Austin, built to support modular slash commands, downloadable packs, and future integration with AnderOS. It’s fast, customizable, and founder-coded from the ground up.

Key Features

  • Slash command interface
  • Pack system for themes and extensions
  • Cross-OS compatibility
  • Future sync with ModTone and BitBoard

Status

Currently in development. First preview coming soon.
